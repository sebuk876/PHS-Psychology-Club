# PHS Psychology Club 🧠

Official website for the **Pasadena High School Psychology Club**, founded in 2025.

## About

The PHS Psychology Club is a student-run organization dedicated to making psychology approachable, relevant, and real. We explore topics ranging from mental health awareness to cognitive biases, social psychology, and the science of habits.

## Getting Started

This is a single-file static website. No frameworks or dependencies required.

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/phs-psychology-club.git
   ```
2. Open `index.html` in any browser

## Deployment

This site is hosted for free via **GitHub Pages**.

To deploy your own version:
1. Fork this repository
2. Go to **Settings** → **Pages**
3. Set source to the `main` branch
4. Your site will be live at `https://yourusername.github.io/phs-psychology-club`
